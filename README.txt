Para poder ejecutar el proyecto debe seguir los siguientes pasos.

1. Descargar el proyecto en eclipse.
2. Ejecutar la clase MVC del paquete main.
3. Seguir los pasos que aparecen en consola.(por defecto se cargan los 3 grafos que se encuentran en la carpeta data)
4. En caso de querer querer cargar otro grafo se debe tener la ruta del archivo y el numero de vertices.(Estos se pediran en la consola)

para ejecutar los algoritmos aparecera un menu en la consola mediante el cual se debe especificar que algotirmo ejcutar dicho menu luce de la siguiente forma

Que algoritmo desea Implementar (escriba el numero)
1. Dijkstra
2. Bellman Ford
3. Floyd Warschall
4. BFS
5. DFS
